P2000 Web User Interface Help

Key Terms

●      P2000 Security Management System: A security system from Johnson Controls, Inc. that can monitor and control the physical security of a facility.

●      P2000 Server: The main computer in a P2000 Security Management System, it runs the P2000 security management software. The current release is version 3.12.

●      P2000 Workstation: A computer running the P2000 workstation software. P2000 workstations allow P2000 operators to monitor and control aspects of the P2000 Security Management System.

●      P2000 Archive Report Server: A computer running the P2000 ARS software; it stores P2000 Server data and generates reports from that data.

●      Microsoft SQL Server Reporting Services (SSRS): Report generation software from Microsoft. The P2000 ARS uses the capabilities of SSRS to create and save reports in XLS and PDF formats.

●      report: A gathering of information about a particular subject. Reports created by the P2000 ARS can be viewed on-screen, printed, exported to a file, saved to a network share, or emailed. A wide variety of report types are available on the P2000 ARS.

●      administration tool: A local tool on the P2000 ARS system that lets you configure and manage the P2000 ARS.

●      live data: Data pulled to a P2000 ARS from a version 3.12 or greater P2000 Server for storage and reporting. You can store live data for 30 years or more and generate a wide variety of reports from it.

●      archived database: A backup of P2000 Server data that can be archived on a P2000 ARS for historical purposes; it can also be restored by a P2000 ARS so that reports can be generated from the data. You can create archived databases using P2000 Servers version 3.12, 3.11, 3.10, and 3.8; additionally, a P2000 ARS can also created archived databases of its data.

●      legacy P2000 Server: An older P2000 Server whose archived databases can be transferred to and then restored by a P2000 ARS; reports can then be created from the data in the restored database. Legacy P2000 Servers include versions 3.11, 3.10, and 3.8.

●      ODBC data connection: The method of exchanging data between a P2000 ARS and a P2000 Server.

●      Web server: A system that delivers Web content. A P2000 ARS uses the P2000 Web UI for running and scheduling reports; the P2000 Web UI can be installed on a P2000 v3.12 Server or on a separate Web server.

●      Web browser: A software application for retrieving and displaying Web content. The P2000 Web UI is accessed from a Web browser.
