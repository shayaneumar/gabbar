Report Appearance

All report files, whether run or scheduled, share certain characteristics:

●      Report title: The name of the report.

●      Filter criteria and values: The criteria and values applied to the data to create the report. These vary based on the selected report.

●      Sort order: The order in the data is being presented.

●      Report data: The data the matches the selected criteria and values. Depending on the selected report and criteria, the display of report data can be many pages long.

●      Report footer: The date the report was run and the number of pages; shown on the bottom of each page of the report.

 

Preliminary – This information may change.
