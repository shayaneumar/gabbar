Running Versus Scheduling Reports

You can either run a report once or schedule a report:

●      Run a report: One time only. When you run a report, you open the finished report using an application that supports the selected file format, XLS or PDF. While the report is open, you can save it for future reference. To run a report, click the Run Now button next to the name of the desired report.

●      Schedule a report: Recurring basis. When you schedule a report, it runs automatically based on the criteria you select. You can configure a scheduled report to be saved to a network share or emailed to one or more addresses. To schedule a report, click the Schedule button next to the name of the desired report.

Both standard and custom reports can be run once or scheduled.
