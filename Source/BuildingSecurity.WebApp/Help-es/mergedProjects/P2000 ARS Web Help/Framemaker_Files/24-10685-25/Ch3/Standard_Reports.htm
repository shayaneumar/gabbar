Standard Reports

The P2000 ARS comes with a set of default standard reports. If necessary, you can configure custom reports and use those as well.

The standard reports are:

●      Alarm Activity: Lists all alarms that were pending during a specified period of time.

●      Alarm History: Lists all alarms that occurred during a specified period of time.

●      Alarm Summary: Shows a column/grid matrix of alarms based on specified criteria.

●      Area Transaction History: Lists the transactions that occurred in a specific area during a specified period of time.

●      Audit Trail: Lists audit items for a specific user during a specified period of time.

●      Entity Transaction History: Lists the transaction history for the specified entity, such as a cardholder.

●      Guard Tour Transaction History: Lists the transaction history for the specified guard tour.

●      Intrusion Transaction History: Lists the intrusion transaction history for a specific intrusion area during a specified period of time.

●      Muster Analysis: Lists the personnel who were in a muster zone during a specified period of time.

●      Transaction History: Lists all transactions on the system during a specified period of time.

 

Preliminary – This information may change.
