P2000 Web User Interface Help

Getting Started

Perform the following series of steps to get started:

●      P2000 Administrators: Assign the appropriate permissions to the operators who will use the P2000 Web User Interface (UI). Refer to the P2000 Software User Manual (Part No. 24-10685-157) for more information on configuring user permissions.

●      Verify your computer meets or exceeds the minimum system requirements (see Verifying the System Requirements); see also Browser Limitations.

●      Log on to the P2000 Web UI software (see Logging On)
