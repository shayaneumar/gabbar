P2000 Web User Interface Help

P2000 Web UI Navigation Options

The P2000 Web UI provides the following navigation options at the top of the screen:

Home – Opens the Alarm Manager page.

Investigate – Opens Investigate page, where users with proper permissions can run reports. Refer to the associated help section or the P2000 Archive Report Server User Manual (Part No. 24-10685-25) for more information.

System Setup – Provides system administration options where users with proper permissions can configure report server configuration settings, configure alarm display options, and upload a company logo.

 

Preliminary – This information may change.
