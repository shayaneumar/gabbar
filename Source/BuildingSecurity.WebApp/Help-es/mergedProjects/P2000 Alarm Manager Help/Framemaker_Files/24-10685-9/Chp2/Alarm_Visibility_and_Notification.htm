Alarm Visibility and Notification

When an alarm occurs, the P2000 Alarm Manager displays the alarm information in columns and rows on the Alarm Manager page, and notifies the Security Operator both visually and audibly (see Alarm Color and Sound Indicators).

With the potential to have a large number of alarms on the Alarm Manager page, the P2000 Alarm Manager categorizes them effectively, so you can quickly:

●      See the local date and time of the alarm (Date/Time column), even when in a different geographical time zone, so you know exactly when the alarm occurred. You can change the local time zone setting within the P2000 Alarm Manager, if desired. See Changing the Time Zone Setting for instructions on changing the time zone setting.

●      Visually and audibly distinguish the alarm’s priority level (Priority column), so that alarms do not go unnoticed and you can promptly respond to the alarm.

●      Identify the current status of the alarm (Alarm Status column), so you can see the location of the alarm within the alarm life cycle and act accordingly. See Alarm Life Cycle for more information.

●      View the source of the alarm (Source column), so you can identify the point of alarm and the elements that caused it (that is, where the alarm occurred, the device associated with the alarm, and so on).

●      See the date and time when an operator last performed an action for the alarm (Action Date/Time column).
