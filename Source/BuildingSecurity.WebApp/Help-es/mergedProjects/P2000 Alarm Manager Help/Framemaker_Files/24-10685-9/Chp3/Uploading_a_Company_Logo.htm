P2000 Web User Interface Help

Uploading a Company Logo

Users with proper permissions can add a company logo (or any other digital graphic) to the upper right corner of the P2000 Web UI screen.

By default, the upper right corner of the screen shows a gray rectangle that says Client Logo. The graphic you add appears in place of the gray rectangle.

Use one of the following required graphic formats for the image: JPG, GIF, or PNG. These formats provide high quality, low size images and are supported by a wide variety of Web browsers.

Note:   Johnson Controls recommends a transparent background for your image. This ensures that the background of your image does not clash with the background of the Web UI screen.

Recommended dimensions for the image are 175 by 65 pixels. Maximum dimensions are 350 by 80 pixels without any resizing of the image, but the file size will also increase. If the image exceeds the maximum dimensions, it will be proportionally resized to fit.

To upload a company logo to the P2000 Web UI screen:

1.     Prepare the image you want to use, ensuring it meets size recommendations and format requirements.

2.     From the menu bar, select System Setup > Upload Company Logo.

The Upload Company Logo page appears.

3.     Click Choose File, locate and select the image you wish to use, and click Open.

The Upload Company Logo page refreshes and displays the selected image.

4.     If the image appears correctly, click Apply.

The image appears in the upper right corner of the screen.

5.     If the image does not appear correctly, stop the procedure, make the necessary changes, and then start the procedure again.
