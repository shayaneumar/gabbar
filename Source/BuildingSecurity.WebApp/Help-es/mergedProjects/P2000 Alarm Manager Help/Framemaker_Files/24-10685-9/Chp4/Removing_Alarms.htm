Removing Alarms

When you remove an alarm from the alarm list, the alarm will no longer be visible to all users; however, users can run the Alarm History report to view information on alarms removed from the list. For more information on running alarm reports, refer to the P2000 Archive Report Server User Manual (Part No. 24-10685-25) or associated online help.

You can remove a single alarm or multiple alarms at once.

To remove a single alarm:

1.     Locate the alarm you wish to remove.

2.     Expand the alarm details by performing one of the following actions:

●      Click the twisty arrow to the left of the alarm.

●      Click the Details/Respond link to the right of the alarm.

3.     Click Remove.

To remove multiple alarms:

1.     Select the check box next to each alarm you wish to remove.

2.     Click drop-down arrow next to the Acknowledge button at the top left of the page and select Remove.
