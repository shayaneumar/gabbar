Sorting Alarms

The P2000 Alarm Manager enables you to sort alarms in the alarm list, so you can more easily find a specific alarm.

You may sort alarms by:

●      Those with a status of Pending

●      Date/Time

●      Priority

●      Alarm Status

●      Source

●      Action Date/Time

For more information on these data columns, see Alarm Visibility and Notification.

You can sort the alarm list in ascending and descending order. For example, sorting by descending date and time lists the alarms in order starting with the most recent date and ending with the oldest date. Conversely, sorting by ascending date and time lists the alarms in order starting with the oldest date and ending with the most recent date.

To sort alarms:

1.     Click the column header of the data that P2000 Alarm Manager will use to sort the alarm list.

To sort the alarm list based on alarms with a status of Pending, click the Pending Alarm icon.

A descending or ascending arrow appears next to the column header to indicate the sorting order.

= Sorted in descending order

= Sorted in ascending order

Note:   The P2000 Alarm Manager does not support multi-attribute sorting. You can only sort by a single attribute at a time.

2.     Click the column header again to toggle between ascending/descending order.
