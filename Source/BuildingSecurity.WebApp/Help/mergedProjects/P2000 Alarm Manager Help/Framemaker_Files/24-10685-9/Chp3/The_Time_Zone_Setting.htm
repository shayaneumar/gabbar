The Time Zone Setting

The Time Zone setting controls the time/date information you see about an alarm in the P2000 Alarm Manager. You can change time zones to change the time/date information you see.

Changing the time zone setting in Alarm Manager does not affect the time/date information for an alarm that is set by the P2000; it changes what you see in Alarm Manager.

The default P2000 Alarm Manager time zone setting is the same as that of the web server hosting the P2000 Web UI.

If your P2000 Alarm Manager is on a web server in the Pacific time zone and your P2000 is in the Eastern time zone, you might want to set the time zone of Alarm Manager to the Eastern time zone so that you know the local time where the alarms are occurring.

To change the time zone setting:

1.     Click the gear icon next to the text that displays the number of total and pending alarms.

2.     Select a time zone from the drop-down list.

The Alarm Manager page refreshes to display the alarms with updated time stamps based on the new time zone selected. The Alarm Manager also displays time stamps for all alarm details (Action Date/Time) based on the new time zone.
