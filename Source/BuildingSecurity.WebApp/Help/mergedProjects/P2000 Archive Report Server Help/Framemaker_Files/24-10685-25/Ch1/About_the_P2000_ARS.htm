About the P2000 ARS

The P2000 ARS archives data and lets you generate detailed reports from that archived data. It provides the reporting tools essential for reviewing responses to alarms, transactions, and audit messages, as well as mustering and guard tour information. It also supports the long-term archiving of data that may be required for government and regulatory compliance.

A P2000 ARS gathers live data from one or more P2000 v3.12 Servers; it can generate reports from that data and archive the data for up to 30 years.

A P2000 ARS can also store report data backups made from P2000 Servers (Versions 3.12, 3.11, 3.10, and 3.8) or from a P2000 v3.12 ARS. Report data backups can be restored by a P2000 ARS and reports generated from the restored data.

The P2000 ARS includes the P2000 Web UI for running and scheduling reports. The P2000 Web UI is installed separately from the P2000 ARS; it can be installed on a P2000 Server or on a corporate web server.
