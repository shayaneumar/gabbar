P2000 Web User Interface Help

Important Terms

●      P2000 Security Management System – A security system from Johnson Controls, Inc. that can monitor and control the physical security of a facility.

●      P2000 Server – The main computer in a P2000 Security Management System; it runs the P2000 security management software.

●      P2000 Archive Report Server – A computer running the P2000 ARS software; it stores P2000 Server data and generates reports from that data.

●      P2000 Web UI – Web-based software used to access data on a P2000 ARS to run and schedule reports. You access the P2000 Web UI using a Web browser.

●      Microsoft SQL Server Reporting Services (SSRS) – Report generation software from Microsoft; a feature of Microsoft SQL Server. The P2000 ARS uses the capabilities of SSRS to create and save reports.

●      Report – A gathering of information about a particular subject. Reports created by the P2000 ARS can be viewed on-screen, printed, exported to a file, saved to a network share, or emailed. The P2000 ARS includes standard reports and also supports custom reports.

●      Administration tool – A local tool on the P2000 ARS system that lets you configure and manage the P2000 ARS.

●      Live data – Data pulled to a P2000 ARS from a Version 3.12 or greater P2000 Server for storage and reporting. You can store live data for up to 30 years and generate reports from it.

●      Report data backup – A backup of P2000 Server data that can be stored on a P2000 ARS; it can also be restored by a P2000 ARS so that reports can be generated from the data. You can create report data backups using P2000 Server Versions 3.12, 3.11, 3.10, and 3.8; additionally, a P2000 ARS can also create report data backups of its live data.

●      Legacy P2000 Server – An older P2000 Server whose report data backups can be transferred to and then restored by a P2000 ARS; reports can then be created from the restored data. Legacy P2000 Servers include Versions 3.11, 3.10, and 3.8.

●      Web server – A system that delivers Web content. The P2000 Web UI can be installed on a P2000 Server or on a separate corporate Web server.

●      Web browser – A software application for retrieving and displaying Web content. The P2000 Web UI is accessed using a Web browser.

●      PDF – A file format used to represent documents; originally created by Adobe Systems, Inc. A P2000 ARS can save reports in PDF format.

●      Microsoft Excel – A spreadsheet application and file format; created by Microsoft Corp. A P2000 ARS can save reports in Microsoft Excel format.

●      CSV – A file format that stores tabular data (numbers and text) in a plain-text format. A P2000 ARS can save reports in CSV format.

●      Network file share – A location on an internal network where files are stored so that they can be accessed by appropriate personnel. A P2000 ARS can save scheduled reports to a network file share.

●      Email – Electronic mail. A P2000 ARS can email scheduled reports to one or more email addresses.
