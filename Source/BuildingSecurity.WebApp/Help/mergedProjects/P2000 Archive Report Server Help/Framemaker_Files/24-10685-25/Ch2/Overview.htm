P2000 Web User Interface Help

Overview

Before you can run or schedule a report using the P2000 Web UI, you must:

●      Install and configure one or more P2000 V3.12 Servers. Refer to P2000 Software User Manual for more information.

●      Install and configure a P2000 ARS. Refer to P2000 Archive Report Server Administrator Guide for more information.

●      Install the P2000 Web UI on a P2000 Server or on a web server. Refer to P2000 Web UI User Guide for more information.

●      Configure the P2000 Web UI to communicate with the P2000 ARS. Refer to Configuring the P2000 Web UI to Communicate with the P2000 ARS for more information.
