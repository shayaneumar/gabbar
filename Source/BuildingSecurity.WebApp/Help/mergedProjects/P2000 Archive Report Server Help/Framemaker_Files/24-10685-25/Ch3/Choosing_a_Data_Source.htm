Choosing a Data Source

The P2000 ARS can run a report against live data that is being gathered from one or more P2000 v3.12 Servers or against report data backups. Scheduled reports are only run against live data.

To run a report against a report data backup, the backup file must be stored on the P2000 ARS and it must restored by the P2000 ARS Administration tool; refer to the P2000 Archive Report Server Administrator Guide for more information.

If more than one data source is available, you need to specify a data source when you create the report.

In the Which data source would you like to use? field, either accept the Default setting (which means the report is run against live data) or pick a restored, report data backup from the drop-down list. Report data backups are listed using their friendly name. There is only one live data source; it is always listed as Default.

Tip:   If you need to run the same report against live data and against a report data backup, you will need to run the report twice.
