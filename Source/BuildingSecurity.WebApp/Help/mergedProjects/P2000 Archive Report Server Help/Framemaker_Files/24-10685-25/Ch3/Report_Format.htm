Report Format

The P2000 ARS supports the following file formats for reports:

●      Microsoft Excel (XLS)

●      PDF

●      CSV

When you run a report, it is automatically saved in the specified format. Additionally, you are prompted to open the report using an application that can display the selected file format; opening the report is optional. When you schedule a report, the file is emailed or saved to a network file share in the selected file format.

Pick the desired file format during report setup.

Note:   You can save a report as PDF or XLS even if you do not have Microsoft Excel or Adobe Acrobat, respectively, installed on your system. Many applications can open CSV files, including Excel. There are alternatives to Excel and Acrobat available on the Internet that let you open and view report files without having those applications on your system.
