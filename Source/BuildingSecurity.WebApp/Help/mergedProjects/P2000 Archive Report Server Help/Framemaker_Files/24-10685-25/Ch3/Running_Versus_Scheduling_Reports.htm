Running Versus Scheduling Reports

You can either run a report immediately or schedule a report:

●      Run a report: Runs immediately. Reports are saved automatically when run. When you run a report, you can open the finished report using an application that supports the selected file format, PDF, XLS, or CSV. To run a report, click the name of the report you want to run on the Run a Report screen. When you run a report, you must specify a data source (if more than one is available), either live data or a report data backup; the default data source is live data. When you run a report, you can pick from multiple date/time ranges, including monthly, weekly, daily, or custom options.

●      Schedule a report: The report is run on a regular basis. When you schedule a report, it runs automatically based on the criteria you select. You can configure a scheduled report to be saved to a network file share or emailed to one or more addresses. To schedule a report, click the name of the report you want to schedule on the Schedule a Report screen. Scheduled reports are only run against live data; you cannot schedule a report to run against a restored report data backup. When you schedule a report, there are only two date/time ranges available; when you specify a daily report, the data is from the previous day; when you specify a weekly report, the data is from the previous seven days.

Both standard and custom reports can be run immediately or scheduled.
