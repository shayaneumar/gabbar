Configuration Criteria

Configuration criteria for reports you are running immediately or scheduling appear in the How do you want to configure your report? section. Each report has a different set of criteria.

Available criteria are:

●      Language: Specify a language for the report.

●      User Name: Show records for all users (*) or just for the user you specify.

●      Site Name: Show records for all sites (*) or just for the site you specify.

●      Partition Name: Show records for all partitions (*) or just the partition you specify.

●      Item Name: Show records for all items (*) or just the item whose name you specify.

●      Alarm Type: Shows records for all alarm types (<all>) or just the alarm type you specify.

●      Alarm Summary Type: Shows records based on the type of alarm summary that you specify.

●      Alarm Category: Show records for all alarm categories (*) or just the alarm category whose name you specify.

●      Description: Show records for all descriptions (*) or just the description whose text you specify.

●      Audit Type: Show records for all audit types (<any>) or just the audit type you specify.

●      Audit Action: Show records for all audit actions (<any>) or just the audit action you specify.

●      Area Name: Show records for all areas (*) or just areas whose name you specify.

●      Transaction Type: Show records for all transaction types (<all>) or just the transaction type you specify.

●      Guard Tour Name: Shows records for all guard tour names (*) or just for the guard tour name you specify.

●      Last Name: Show records for all last names (*) or just for users with a specified last name.

●      First Name: Show records for all first names (*) or just for users with a specified first name.

●      Middle Name: Show records for all middle names (*) or just for users with a specified middle name.

●      Cardholder ID: Show records for all cardholders (*) or just for users with a specified cardholder ID.

●      Badge Number: Show records for all badge numbers (*) or just for users with a specified badge number.

●      History Type: Show records for all history types (<all>) or just the history type you specify.

●      Terminal Name: Show records for all terminal names (*) or just for a specified terminal name.

●      Elevator/Cabinet Option: Show records for elevators or cabinets, as specified. Select <Not Applicable> if you do not want to filter results based on elevators or cabinets.

●      Elevators/Cabinets: Show records for all elevators or cabinets (*) or just for specified elevators or cabinets.

●      Floors/Doors: Show records for all floors and doors (*) or just for specified floors and doors.

●      Intrusion Name: Show records for all intrusion names (*) or just for specified intrusion names.

●      Date/Time Range: Specify a time frame for the report. Select Custom, Last 7 Days, Last Month, Last Week, or Yesterday from the drop-down list. If you select Custom, you need to select a Date/Time From and a Date/Time To in the following two fields.

●      Date/Time From: Specify the beginning of a custom time range. When you click in the field, a calendar appears to help you select the appropriate day. Only available if your Date/Time Range selection is Custom.

●      Date/Time To: Specify the end of a custom time range. When you click in the field, a month calendar appears to help you select the appropriate day. Only available if your Date/Time Range selection is Custom.

●      Sort Order Option: Select the criteria by which the report data is sorted: Date Time, Item Name, or User Name.

●      Order Sequence: Select whether the report data is sorted in Ascending or Descending order.

●      Muster Zone Name: Shows records for all muster zone names (*) or just for the muster zone name you specify.

●      Group Type: Shows records for all muster zone group types (<any>) or just for the muster zone group type you specify.

●      Processor Mode: Shows records for all muster zone processor modes (<any>) or just for the muster zone processor mode you specify.

●      Execution mode: Shows records for all muster zone execution modes (<any>) or just for the muster zone execution mode you specify.
