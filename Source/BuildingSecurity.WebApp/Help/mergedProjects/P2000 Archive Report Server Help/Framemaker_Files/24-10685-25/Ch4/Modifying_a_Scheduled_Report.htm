Modifying a Scheduled Report

Existing scheduled reports can be modified or deleted. You can also modify a scheduled report and then saved it as a new report.

To modify or delete a scheduled report:

1.     Log on to the P2000 Web UI, click Reporting, and then slide down and select Manage Scheduled Reports.

The Manage Scheduled Reports screen appears, listing all currently scheduled reports.

2.     To delete a report, locate the report you want to delete and click the x to the left of the report name.

The scheduled report is deleted.

3.     To modify a report, locate the report you want to modify and click the report name.

The Modify [report name] screen appears, showing the current settings for the scheduled report.

4.     Make the desired changes to the settings of the scheduled report.

5.     When you are done making changes to the settings, click:

●      Update Scheduled Report to save the changes to the report. A summary screen appears, showing the new settings for the scheduled report. Click Modify This Scheduled Report to return to the Modify [report name] screen.

●      Save as New Report to save the new configuration as a separate scheduled report. Click Modify This Scheduled Report to return to the Modify [report name] screen.

Note:   If you are saving a scheduled report as a new report, you may want to enter a new description for the report in the Scheduled Report Description field so that you can tell the two reports apart.
