Scheduling a Report

Scheduling a report means to set it up to run automatically for as many times as specified.

To schedule a report:

1.     Log on to the P2000 Web UI, pull down the Reporting menu, and select Schedule a Report.

2.     On the Schedule a Report screen, select Standard or Custom, then click on the name of the report you want to schedule.

The [report name] page for the report you selected appears.

3.     Enter a friendly name for your scheduled report in the How do you want to describe your report? field.

4.     Specify criteria or accept the default criteria for the configuration of your report in the How do you want to configure your report? field.

Note:   Each report only uses a subset of the available criteria. Also, some criteria appear dimmed if they do not apply to the current selection.

Choose from the available criteria (see Configuration Criteria for a list of criteria) for the report you are scheduling.

5.     Choose to generate your report as a PDF, Microsoft Excel (XLS), or CSV file by selecting the appropriate button in the In what format would you like to receive your report? field.

6.     Specify when you want your report to run in the When do you want to run your report? field:

●      Select Daily to get a daily report; you’ll need to specify the time the report should be run, a date to start running the report in the Starting on field, and an optional end date in the Ending on field.

●      Select Weekly to get a report once a week; you’ll need to specify the day and time the report should be run, a date to start running the report in the Starting on field, and an optional end date in the Ending on field.

7.     Specify what you want to do with your scheduled report in the Where do you want to receive your report? field:

●      To save the report to a network file share, select File Share, enter a valid path in the Location field, then enter a valid User Name and Password if these are needed to save the file to the specified location.

●      To email the report, select Email, enter one or more email addresses in the Email address(es) field, enter text to go into the Subject of the message, and finally enter text in the Comment field to go into the body of the message.

8.     Click Schedule Report.

After the report is generated, it is emailed or saved to the specified network file share in the specified format.
