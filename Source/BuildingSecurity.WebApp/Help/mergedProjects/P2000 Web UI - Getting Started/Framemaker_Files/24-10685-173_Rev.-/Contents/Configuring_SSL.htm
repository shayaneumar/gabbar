P2000 Web User Interface Help

Configuring SSL

Communications between the P2000 Web UI and clients can be encrypted using Secure Sockets Layer (SSL), a protocol designed to prevent certain data security risks. You should establish secure communications to the web server before using Web UI for the first time.

Secure communications are achieved through the use of certificates and encryption keys. 

To establish encrypted communications between the P2000 Web UI and clients., you will use a self-signed SSL certificate file, P2000WebUI.crt. You can select to create this file on the P2000 server or on an external drive during the P2000 software installation. Following this, you need to distribute the public SSL key to all clients from which you plan to access the web server.

Note:   A different procedure is required to secure communication between the P2000 Web UI and the P2000 Archive Report Server (ARS). For details, see Communications with the P2000 ARS.

 

Overview of SSL Self Signed Certificate Implementation

Important:   For security reasons, do not distribute the pubic key for the SSL certificate over the network.

To distribute the public SSL key:

1.     Copy the public key for the SSL certificate, P2000WebUI.crt, from the following location on the web server (the server where you installed the P2000 Web UI) to a flash drive:

C:\Program Files (x86)\Johnson Controls

Note:   Some systems use “Program Files” instead of “Program Files (x86).”

2.     Transfer the file to the client computer.

Once the public key is on the client computer, you need to install it. The procedure depends on the browser you are using. Use the following instructions as general guidelines.

To install SSL certificate in Internet Explorer®, Safari®, or Chrome:

1.     Double-click the public key file P2000WebUI.crt to open it.

2.     Follow prompts to install it into a trusted key store.

3.     When prompted for location, select the following store on the local machine: Trusted Root Certification Authorities.

4.     Follow prompts to complete the installation.

To install SSL in Firefox:

1.     Start Firefox and go to Options>Options.

2.     In the Options window, click Advanced and then click Encryption.

3.     Click View Certificates. The Certificate Manager window opens.

4.     Under Servers tab, click Import...

5.     Select your certificate and click Open.

6.     In the Certificate Manager window, find the certificate you just imported (it is not yet trusted). Select it from the list and click Edit Trust... 

7.     In the Edit Website Certificate Trust Settings window, select Trust the authenticity of this certificate. Click OK.

8.     Close all the windows.
