P2000 Web User Interface Help

P2000 Web UI Navigation Options (Menu Bar)

P2000 Web UI provides several navigation options. In most cases, except for Help and User (Sign Out), when you click an option, the P2000 Web UI replaces the page on the current tab. To open the page on a new tab, press and hold the Ctrl or Shift key (depending on the browser) when you click.

The P2000 Web UI provides the following navigation options at the top of the screen:

Note:   Some options may not be available based on the permissions assigned to your account by the P2000 administrator.

Alarm Manager – Opens the Alarm Manager page.

Reporting – Allows users with proper permissions to run or schedule reports, or manage report subscriptions. Refer to the associated help section or the P2000 Archive Report Server User Manual for more information.

System Setup – Provides system administration options where users with proper permissions can configure report server configuration settings and alarm display options.

Help – Launches the P2000 Web UI online help.

User (Sign Out) – Displays the name of the signed-in user and allows you to sign out.
