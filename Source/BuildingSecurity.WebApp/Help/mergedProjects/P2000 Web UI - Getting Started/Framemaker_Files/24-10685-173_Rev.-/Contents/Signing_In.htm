P2000 Web User Interface Help

Signing In

When signing in to the P2000 Web UI, note the following:

●      The User Name field is not case-sensitive.

●      The Password field is case-sensitive.

●      You need appropriate permissions, assigned by the system administrator, to use the P2000 Web User Interface (UI).

●      You have a limited number of attempts to correctly sign in. If you exhaust the number of attempts, the system locks your account. Only a user with proper permissions, such as a P2000 administrator, can unlock your account.

●      If you successfully sign in, but you cannot use specific applications or see any alarms, you may not have access permission to these applications. Consult with the P2000 administrator for assistance.

Important:   If receive a warning saying “There is a problem with this website's security certificate” (or the equivalent, depending on the browser), do not proceed. You must configure SSL settings prior to connecting to the web server. See Configuring SSL for instructions.

To sign in to the P2000 Web UI:

1.     Launch a browser instance.

2.     Navigate to the following address:

https://[IP address or computer name of web server]/p2000webui/

For example: https://127.0.0.2/p2000webui/

3.     In the Sign In page, enter your user name and password, and click Sign In.

4.     Depending on your user permissions, Alarm Manager page or the Run a Report page appears.
