P2000 Web User Interface Help

Uploading a Company Logo

Users with proper permissions can add a company logo (or any other digital graphic) to the P2000 Web UI login screen.

We recommend a transparent background for your image. The logo file must be in PNG format and must be named logo.png. Limit the size of the logo to 200 pixels wide by 100 pixels high.

To upload a company logo to the P2000 Web UI screen:

1.     Prepare the logo.png image, ensuring it meets size recommendations and format requirements.

2.     Copy the image file to the following directory. If the folder does not exist, create it manually:

C:\Program Files (x86)\Johnson Controls\P2000WebUiData

Note:   Some systems use “Program Files” instead of “Program Files (x86).”
